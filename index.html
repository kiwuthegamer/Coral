Coral is a text editor that predicts text and corrects spellings. I programmed Coral using HTML (Hypertext Markup Language), CSS (Cascading Style Sheets), and JS (JavaScript). I coded it using almost no external help and all of the code is entirely original. What skills did I gain from working on Coral? Throughout this project I gained these skills: Programming Skills - While coding, I had to learn many new methods of completing a given task. Design Skills - I had to design the layout, colours and animations on the website and then implement them in code. Unfortunately, there will be many bugs, because no code is perfect. I started this project with much less programming experience than I have now.
